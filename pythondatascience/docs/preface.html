Code

Code
print("Python")
###> Python

在 Rstudio Console 中交互式运行 Python

  • exit / quit 退出 Python 环境

Code
library(reticulate)
py_config()

reticulate::repl_python()

# 写在前面 {.unnumbered}

在文中使用的集成开发环境（IDE）是Rstudio，`code` 表示代码（code）

R代码块中， `#>` 表示运行R代码输出的结果，如

```{r r-code}
#| comment: "#>"
print("R")
```

Python代码块中， `#>>>` 表示运行Python代码输出的结果，如

```{python python-code}
print("Python")
```

## **在 R**studio **Console 中交互式运行 Python**

-   `reticulate::repl_python ()` *read-eval-print-loop* (REPL)

-   `exit / quit` 退出 Python 环境

```{r}
#| comment: "#>"
#| eval: false
library(reticulate)
py_config()

reticulate::repl_python()
```
